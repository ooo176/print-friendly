# Press 打印主题设计方案（精简版）

## 设计原则

### 黑白主题（B&W Optimized）
- **目标**：节省墨水，清晰易读，适合日常打印
- **特点**：纯黑白或灰度，无彩色元素，高对比度
- **适用场景**：办公文档、学习资料、存档备份

### 彩色主题（Color Optimized）
- **目标**：视觉美观，情感表达，适合展示和收藏
- **特点**：丰富配色，装饰元素，品牌感强
- **适用场景**：演示文稿、作品集、礼物卡片

---

## 黑白主题系列（5个）

### 1. Minimalist（极简主义）✓ 已实现
**灵感**：现代极简设计
- **字体**：Helvetica / 苹方
- **特点**：
  - 大量留白
  - 无衬线字体
  - 标题超大，正文小
  - 单色，无装饰
- **适合**：演讲稿、要点总结、快速浏览

### 2. Classic（经典书籍）
**灵感**：传统纸质书籍排版
- **字体**：Garamond / 宋体
- **特点**：
  - 衬线字体，优雅易读
  - 首字下沉（drop cap）
  - 页眉页脚带页码
  - 章节标题居中，装饰线
- **适合**：长文阅读、小说、散文

### 3. Academic（学术论文）
**灵感**：IEEE/ACM 论文格式
- **字体**：Times New Roman / 中易宋体
- **特点**：
  - 双栏排版（可选）
  - 标题编号（1.1, 1.2）
  - 引用标注 [1]
  - 页边距窄，信息密度高
- **适合**：研究论文、技术文档、学术资料

### 4. Newspaper（报纸专栏）
**灵感**：传统报纸排版
- **字体**：Georgia / 仿宋
- **特点**：
  - 多栏窄版（2-3栏）
  - 栏间分隔线
  - 标题粗黑，正文紧凑
  - 首行缩进
- **适合**：新闻文章、专栏评论、信息密集内容

### 5. Technical（技术手册）
**灵感**：程序员文档/API手册
- **字体**：Roboto Mono / 等线
- **特点**：
  - 代码块突出显示
  - 侧边栏目录
  - 参数表格样式
  - 警告/提示框
- **适合**：技术文档、编程教程、API文档

## 彩色主题系列（5个）

### 6. Notebook（笔记本）✓ 已实现
**灵感**：手写笔记本
- **配色**：浅蓝横线 + 红色边线
- **字体**：Lora / 思源宋体
- **特点**：
  - 横线背景
  - 左侧红线
  - 螺旋装订孔
  - 手写字体标题
- **适合**：学习笔记、读书摘要、个人整理

### 7. Naruto（火影忍者）✓ 已实现
**灵感**：动漫主题
- **配色**：橙色 + 黑色
- **字体**：特色字体
- **特点**：
  - 主题背景水印
  - 动漫风格装饰
  - 彩色标题
- **适合**：粉丝内容、主题收藏

### 8. Sakura（樱花）
**灵感**：日式和风美学
- **配色**：樱花粉 + 米白 + 深灰
- **字体**：Noto Serif JP / 思源宋体
- **特点**：
  - 樱花花瓣水印
  - 和纸纹理背景
  - 竖排标题（可选）
  - 印章样式装饰
- **适合**：文学作品、诗歌、日式内容

### 9. Ocean（海洋）
**灵感**：深海宁静感
- **配色**：深蓝 + 青绿 + 白色泡沫
- **字体**：Merriweather / 思源宋体
- **特点**：
  - 蓝色渐变背景
  - 波浪装饰线
  - 引用框带阴影
  - 清爽配色
- **适合**：旅行游记、散文随笔、冥想内容

### 10. Sunset（日落）
**灵感**：黄昏暖色调
- **配色**：橙红 + 金黄 + 紫粉渐变
- **字体**：Playfair Display / 方正书宋
- **特点**：
  - 暖色渐变背景
  - 金色标题
  - 柔和阴影
  - 温馨氛围
- **适合**：生活随笔、情感文章、温暖故事

### 设计要点

**黑白主题设计要点：**
- 使用 `#000` 纯黑和 `#fff` 纯白
- 灰色仅用于辅助信息（`#666`, `#999`）
- 避免大面积深色背景（浪费墨水）
- 打印时自动移除装饰性元素

**彩色主题设计要点：**
- 主色不超过 3 种
- 背景色浅淡（避免大面积深色）
- 提供"彩色打印"开关
- 打印时可选黑白模式
